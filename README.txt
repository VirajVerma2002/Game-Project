To run the game simply open the application file.

To begin the game press Enter as prompted.

To control the spaceship use the mouse/trackpad and the spaceship will move along the x-axis relative to the cursor.

The score will update by 50 points for each row of asteroids dodged.

You will begin with 3 lives which will increase by 1 every 200 points.

For each time the spaceship collides with an asteroid 1 life will be deducted as well as 20 points.

The speed(difficulty) will increase for each row of asteroids dodged.

To replay simply press Enter as prompted.
